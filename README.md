### 🛠️ Installation

#### Navigate into the project directory

```
cd release-please-practice
```

#### Install dependencies

```
yarn
```

### ▶️ Running the App

#### To start the app in development mode:

```
yarn run dev
```

To start the app in production mode:

* Build the project

  ```
  yarn run build
  ```

* Run app in production

  ```
  yarn run start
  ```
